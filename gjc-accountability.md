# GJC Accountability System

Simon's request: Keep him on track and accountable for GJC progress without being annoying.

## 📅 Daily (Weekday Morning Briefs)
Include GJC status:
- Tasks due today (from Notion Daily To Dos, filtered by GJC tag)
- Overdue tasks (highlight if >3 days overdue)
- If a task is stuck: Ask "What's blocking this?"
- Celebrate yesterday's completed GJC tasks

## 📅 Weekly (Monday 8:00 AM ET)
Full GJC check-in via Telegram:
1. **Wins this week** - Celebrate completed tasks
2. **What's stuck** - Identify blockers
3. **Top 3 priorities** - Focus for this week
4. **Next action** - Specific recommendation to move forward

Review progress through phases:
- Phase 1: Organic Build (TikTok/IG content, brand assets)
- Phase 2: Reviews & Testimonials (friend orders, video testimonials)
- Phase 3: Emails (Omnisend welcome flow, campaign templates)
- Phase 4: Paid Ads (Creator PR boxes, Facebook ads)

## 🔔 Proactive Nudges (As Needed)
When to nudge:
- **Task slips >3 days:** "Hey, [task] has been on your list for a few days. What's blocking it?"
- **Dependency clears:** "Website launched! This unblocks: friend orders, testimonials, email setup."
- **Spinning on content:** "You've edited this video 3 times. Ship it and iterate. Done > perfect."
- **Friday follow-up:** If Simon commits to something specific ("I'll finish X this week"), check in Friday afternoon

## 🎯 Tone & Style
- Direct, not hand-holding
- "What's blocking this?" not "Let me know if you need help"
- Celebrate wins genuinely
- Recommend next actions, don't just report status
- Default to "ship it, iterate later" for content decisions
